# Chatty Frontend

React 19 приложение для чата с использованием Socket.io

## Установка

```bash
pnpm install
```

## Разработка

```bash
pnpm dev
```

Приложение будет доступно на http://localhost:3000

## Сборка

```bash
pnpm build
```

## Переменные окружения

Создайте файл `.env` на основе `.env.example`:

```
VITE_SOCKET_URL=http://localhost:4000
```

## Технологии

- React 19
- Vite
- Socket.io Client
- CSS Modules
